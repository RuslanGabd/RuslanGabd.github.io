Normalization

Normalization is the process of organizing data in a database to reduce redundancy and improve consistency. There are three main normal forms: First normal form removes repeating groups and ensures each column contains atomic values. Second normal form removes partial dependencies, meaning all columns must depend on the whole primary key. Third normal form removes transitive dependencies, so non-key columns depend only on the primary key. In practice, normalization helps prevent data duplication and makes updates safer and easier.
